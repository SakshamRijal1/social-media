# Social Media Platform 🚀

> ⚠️ This project is currently under active development.

A full-stack social media application built with Node.js, Express.js, MongoDB, Clerk Authentication, Inngest, ImageKit, and React.

## Current Status

### ✅ Completed
- User Authentication (Clerk)
- User Profile Management
- Image Upload (ImageKit)
- MongoDB Integration
- Connection/Follow System
- Background Jobs with Inngest
- Email Notifications with Nodemailer & Brevo

### 🚧 In Progress
- Post Creation System
- Like System
- Feed Optimization

### 📋 Planned Features
- Comments
- Real-time Chat
- Notifications
- Search Users
- Saved Posts
- Stories

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Clerk
- Inngest
- ImageKit
- Nodemailer
- Brevo
- React
- Tailwind CSS

## Installation

```bash
git clone <your-repository-url>
cd social-media-app
npm install
npm run dev
```
